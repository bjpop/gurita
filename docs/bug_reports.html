Bug reports

Please submit bug reports using GitHub’s issue tracker.

When submitting a bug report please include the following information where possible:

  • The version of hatch used, this can be found with the --version command line argument.

  • The exact command line argument that caused the bug to occur.

  • A description of the input data that caused the bug to occur. Ideally this would be a small example data set that causes the bug to occur.

  • What went wrong.

  • What you expected to happen.
