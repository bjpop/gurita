heatmap

Heatmap showing the relationship between two categorical features and a numerical feature.

hatch heatmap <arguments>

Heatmap plots are based on Seaborn’s heatmap library function.

Argument

Description

Reference

-h

display help

Simple example

Heatmap showing the number of passengers by month and year in the flights.csv data set:

hatch heatmap -y year -x month -v passengers -- flights.csv

The output of the above command is written to flights.month.year.passengers.heatmap.png:


Getting help

The full set of command line arguments for heatmap plots can be obtained with the -h or --help arguments:

hatch heatmap -h
